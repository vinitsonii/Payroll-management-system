# Payroll Management System

A Payroll Management System developed using PHP and MySQL for managing employee records, salary processing, attendance tracking, and payroll operations efficiently.

The system provides a centralized and database-driven platform for handling payroll-related tasks through a clean and user-friendly interface.

---

## Features

- Employee management
- Salary calculation and processing
- Attendance tracking
- Payroll management
- Admin dashboard
- CRUD operations
- Database integration
- Responsive user interface

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

---

## Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/vinitsonii/Payroll-management-system.git
```

2. Move the project folder to your local server directory

Example:
- XAMPP → `htdocs`
- WAMP → `www`

3. Create a MySQL database

4. Import the provided `.sql` file using phpMyAdmin

5. Configure database connection settings

6. Start Apache and MySQL services

7. Open the project in browser

```text
http://localhost/Payroll-management-system
```
